# Language Learner - Obsidian 语言学习插件

**版本**: 0.3.3
**作者**: the_tree
**最低 Obsidian 版本**: 0.12.17
**许可证**: MIT

## 项目简介

Language Learner 是一款功能全面的 Obsidian 语言学习插件，集阅读、查词、复习、使用、统计于一体。该插件专为语言学习者设计，提供从词汇查询到长期复习的完整学习流程支持。

## 核心特性

### 1. 多词典查词系统

- **弹出式查词**: 可在插件设置中快速开启/关闭弹出式查词功能
- **多词典支持**: 集成有道词典、剑桥词典、句酷双语、海词词典、DeepL 翻译等多个词典源
- **词典优先级**: 支持自定义各词典的优先级顺序
- **动词形式美化**: 动词的各种变化形式（第三人称单数、过去式、过去分词等）以紧凑美观的方式展示
- **智能单词清理**: 划词模式下自动清理多余空格和标点符号，确保查词准确性
- **缩写识别**: 自动识别常见缩写（如 "Mr.", "Dr." 等），保持正确的单词状态
- **API 超时保护**: 词典查询请求 8-10 秒超时机制，避免无限等待

### 2. 中文释义自动填充

- 自动从查词结果中提取中文释义
- 智能过滤动词变化表（复数、第三人称单数、现在分词、过去式、过去分词等）
- 只保留真正的中文释义内容，提高学习效率

### 3. 学习管理面板

- **词汇管理**: 查看、添加、编辑和删除学习词汇
- **标签分类**: 为单词添加标签进行分类管理
- **快速重置**: 一键清空表单，重新开始添加
- **类型检测**: 根据是否包含空格自动判断单词/短语类型
- **复习系统**: 内置复习功能，支持间隔重复

### 4. 数据面板（单词列表）

- **显示/隐藏无视单词**: 可切换显示或隐藏已标记为"无视"的单词
- **批量操作**: 支持批量删除、批量状态更新、批量标签添加
- **高级搜索**: 强大的搜索和过滤功能
- **标签筛选**: 按标签快速筛选词汇

### 5. 搜索面板

- **快捷加入生词本**: 直接在搜索面板将单词添加到学习列表
- **剪贴板集成**: 一键从剪贴板获取单词并查询
- **历史记录**: 支持前进/后退浏览历史记录
- **多词典切换**: 快速在不同词典间切换查看结果

### 6. 右键菜单集成

- **直接添加生词**: 在文章中选择单词后，通过右键菜单直接添加到生词本
- **快速查词**: 右键菜单提供快速查词入口

### 7. 阅读模式

- **智能分页处理**: 自动过滤分页符，防止页面显示空白
- **单词计数统计**: 实时统计阅读过程中的单词数量
- **结束阅读标记**: 阅读结束后一键标记生词
- **笔记功能**: 在阅读过程中随时添加笔记
- **多选支持**: 支持按住 Ctrl/Cmd 键进行不连续选择多个单词/词组
- **键盘导航**: 
  - 方向键（上/下/左/右）翻页
  - Home 键回到首页
  - End 键跳转到末页

### 8. 自定义快捷键

所有主要功能均支持自定义快捷键：
- 打开查词面板
- 打开新词面板
- 打开统计面板
- 打开单词列表
- 翻译选中文本
- 输入查词
- 添加单词到学习列表

### 9. 系统稳定性

- **全局错误处理**: 插件初始化或卸载失败时给出明确提示，不会静默崩溃
- **资源清理优化**: 组件卸载时正确释放图表、定时器等资源，防止内存泄漏
- **生产环境日志静默**: 仅在开发模式下输出调试日志，生产环境不泄露调试信息
- **卸载清理**: 插件退出时清理所有定时器和监听器

## 安装方法

### 方法一：手动安装

1. 下载最新版本的插件文件
2. 将整个 `obsidian-language-learner-0.3.3` 文件夹复制到您的 Obsidian vault 的 `.obsidian/plugins/` 目录下
3. 重启 Obsidian 或重新加载插件
4. 在 Obsidian 设置中找到「社区插件」
5. 找到「Language Learner」插件并启用

### 方法二：从 GitHub 安装（如果已发布）

1. 在 Obsidian 设置中打开「社区插件」
2. 点击「浏览」按钮
3. 搜索 "Language Learner"
4. 点击「安装」并启用

## 使用指南

### 弹出式查词配置

1. 进入 Obsidian 设置页面
2. 找到 Language Learner 插件设置
3. 找到「Popup search」选项
4. 点击开关按钮启用/禁用
5. 启用后，在笔记中选择单词即可弹出查词面板

### 添加单词到学习列表

#### 方式一：使用搜索面板

1. 在搜索面板中输入或查询单词
2. 点击「加入生词本」按钮
3. 系统自动获取中文释义并保存

#### 方式二：从剪贴板添加

1. 复制要学习的单词到剪贴板
2. 在搜索面板点击「从剪贴板添加」按钮
3. 系统自动粘贴并查询该单词

#### 方式三：右键菜单

1. 在文章中选择要学习的单词
2. 右键点击打开上下文菜单
3. 选择「加入生词本」选项

#### 方式四：阅读模式添加

1. 在阅读模式中选择单词
2. 点击查询按钮
3. 在弹出的学习面板中提交

### 单词管理操作

1. 打开「单词列表」面板
2. 使用搜索框或高级搜索过滤单词
3. 使用标签筛选特定类别的词汇
4. 选中多个单词进行批量操作：
   - 批量删除
   - 更新学习状态
   - 添加标签
5. 如需查看已无视的单词，点击「显示无视单词」按钮

### 阅读模式使用流程

1. 创建带有 `langr: true` frontmatter 的笔记
2. 点击标题栏的阅读图标进入阅读模式
3. 选择单词进行查词或添加到学习列表
4. 使用方向键导航：
   - 上/下箭头：翻页
   - 左/右箭头：翻页
   - Home 键：回到首页
   - End 键：跳转到末页
5. 结束阅读后点击「结束阅读」按钮标记生词

## 配置选项说明

### 基础设置

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| native | 母语语言代码 | zh |
| foreign | 外语语言代码 | en |
| popup_search | 是否启用弹出式查词 | false |
| auto_pron | 自动播放发音 | true |
| function_key | 功能键设置 | ctrlKey |

### 词典配置

| 词典名称 | 说明 | 默认优先级 |
|----------|------|------------|
| youdao | 有道词典 | 2 |
| cambridge | 剑桥词典 | 1 |
| jukuu | 句酷双语 | 3 |
| hjdict | 海词词典 | 4 |
| deepl | DeepL 翻译 | 5 |

### 显示设置

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| dict_height | 词典面板高度 | 250px |
| font_size | 阅读模式字体大小 | 32px |
| font_family | 阅读模式字体 | "Times New Roman" |
| line_height | 行高 | 1.8em |
| learn_panel_font_size | 学习面板字体大小 | 0.85em |

### 数据存储设置

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| db_name | 数据库名称 | 笔记库 |
| word_database | 单词数据库文件名 | wordDB.md |
| review_database | 复习数据库文件名 | reviewDB.md |
| col_delimiter | 列分隔符 | , |

## 已知问题与限制

1. 部分词典 API 可能有访问频率限制
2. DeepL 翻译需要有效的 API 密钥
3. 阅读模式目前仅支持带有特定 frontmatter 的笔记
4. 某些特殊字符可能影响划词查词的准确性

## 故障排除

### 常见问题

**Q: 插件无法加载？**
A: 请检查 Obsidian 版本是否满足最低要求（0.12.17），确认插件文件完整性。

**Q: 查词结果显示为空？**
A: 请检查网络连接，确认相关词典 API 是否可用，检查 API 密钥配置。

**Q: 阅读模式无法启动？**
A: 请确保笔记包含 `langr: true` 的 frontmatter 设置。

**Q: 快捷键不生效？**
A: 请在插件设置中检查快捷键配置，确保没有与其他插件冲突。
